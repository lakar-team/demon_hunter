**Demon Hunter** is a fast-paced, experimental action game built on a custom-designed JavaScript game engine. Test your reflexes with a deep combo system and atmospheric pixel-art aesthetics.

## 🕹️ Game Features

- ⚔️ **Combo Engine v5.0**: A frame-perfect combat system that rewards rhythmic attacks and timing.
- ⚡ **High-Octane Gameplay**: requestAnimationFrame-driven loop for smooth, low-latency player movement and collision.
- 🩸 **Status Dynamics**: Real-time health and energy management tracked via a dedicated HUD.
- 🌑 **Void Aesthetic**: Minimalist, high-contrast visual design focused on clarity and action.

## 🚀 Technical Architecture

- **Engine**: Custom procedural JavaScript game loop.
- **Rendering**: Hardware-accelerated HTML5 Canvas.
- **State Management**: Localized object-oriented architecture for player and enemy entities.
- **Optimization**: Minimal asset footprint for instant loading.

## 🛠️ How to Play

1. Open `index.html` in any modern web browser.
2. Use the keyboard to navigate and execute attack patterns.
3. Chain attacks to build your combo meter and clear the field.
